## Comandos de deploy

Na raiz do projeto local:

```powershell
git status
git add .
git commit -m "hotfix aula ia railway schedule analyze-gcs"
git push
```

No Railway, confirme:

- Root Directory: `backend`
- Build Command: `npm install`
- Start Command: `npm start`

Depois teste:

```powershell
Invoke-RestMethod -Uri "https://aulaabertafixed-production.up.railway.app/health"

$body = @{
  test = $true
  bucketName = "dk-aula-aberta-videos"
  fileName = "recordings/teste.mp4"
} | ConvertTo-Json

Invoke-RestMethod `
  -Uri "https://aulaabertafixed-production.up.railway.app/analyze-gcs" `
  -Method Post `
  -ContentType "application/json" `
  -Body $body
```

A resposta esperada para o teste é JSON com `ok: true`, `receivedBucketName` e `receivedFileName`.
